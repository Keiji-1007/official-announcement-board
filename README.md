# 公式発表ボード

政府・国会・官公庁などの公式発表を入力し、解説や意見を加えずに、note記事として読みやすい形式へ整理する静的Webツールです。

## ファイル構成

- `index.html`
- `styles.css`
- `script.js`
- `README.md`

## 使い方

1. `index.html` をブラウザで開きます。
2. 次の項目を入力します。
   - 公式発表タイトル
   - 発表日
   - 出典URL
   - 公式発表本文
3. `記事形式に整える` ボタンを押します。
4. 出力欄に次の項目が整形されます。
   - note記事タイトル
   - 発表日
   - 決定事項
   - 所管または関係機関
   - 本文に書かれている範囲での内容整理
   - 出典URL一覧
   - note見出し画像用プロンプト
   - X投稿文
5. `コピー` ボタンで出力欄の内容をそのままコピーできます。

## 仕様メモ

- 解説を加えない
- 意見を入れない
- 推測しない
- 公式発表本文にない情報を補わない
- 不明な項目は `不明` と表示
- 箇条書きと短い段落で表示
- 出典URL一覧は `媒体名 URL` 形式

## GitHub Pagesで公開する手順

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリ画面で `Settings` を開きます。
3. 左メニューの `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選択します。
5. Branch に `main`（または公開したいブランチ）を選び、フォルダを `/ (root)` にします。
6. `Save` を押します。
7. 数分待つと、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。
